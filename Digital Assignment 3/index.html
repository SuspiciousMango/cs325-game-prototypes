  • High Concept: As the new pirate phaser dude, walk around the map collecting randomly spawned treasure. There are 5 in total, but each chest contains a random one!
  • Theme: I went for the "pirate" theme, even going so far as to resprite the phaser dude with an eyepatch and (almost invisible) hook hand.
  • Mandated Variety: This game uses a direct, keyboard input and a decent amount of randomness. It is a platformer designed to appeal to the collector play style.
  • Prototype Goal: Control via keyboard, "pausing" the game when important events happen, alternating an image between multiple sprites
